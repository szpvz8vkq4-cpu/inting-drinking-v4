# RAFTEL V4.8 — Ultimate Bravery

Version Next.js / Supabase du lobby RAFTEL avec :

- rooms online Supabase
- rôles host / invité / spectateur
- chat
- journal
- roulette
- gages LoL
- module Ultimate Bravery

## Lancer en local

```bash
npm install
npm run dev
```

Puis ouvrir :

```txt
http://localhost:3000
```

## Déployer sur Vercel

Vérifie les variables d'environnement Vercel :

```txt
NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY
```

Puis push GitHub :

```bash
git add .
git commit -m "V4.8 Ultimate Bravery"
git push
```
